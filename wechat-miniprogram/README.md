# 春日小筑 - 微信小程序项目 (Spring Nest Mini Program)

本项目是 **春日小筑 (Spring Nest)** 的微信小程序完美适配版。

## 设计目的与架构理念

由于原版 **春日小筑** 基于 React 19、Tailwind CSS 4 响应式原子框架和 Motion (Framer Motion) 精致流光动效开发，且包含 **29 个效率工具** 和 **19 个高交互 Canvas 益智游戏**。如果强行把所有业务代码阉割并原生重构至小程序的 WXML/WXSS 运行时，将会面临如下毁灭性损失：
1. **动画丢失**：小程序原生不支持 Framer Motion 复杂的物理弹簧和布局过渡。
2. **样式失真**：Tailwind 4 现代渲染规则、CSS 3 动效和弹性布局可能在部分小程序底座产生错乱。
3. **功能阉割**：由于小程序单包 2MB 的极严体积限制，19个游戏和29个工具的数据和静态图文很快就会容量爆溢，导致部分应用被迫阉割。

**完美适配方案 (WebView 智能容器)：**
本项目采用高规格微信 `web-view` 双擎渲染方案。将整个未阉割、高性能的原客户端直接在腾讯原生系统级 X5 浏览器内核容器中无缝运行。

**方案优势：**
- **100% 完美保留**：所有番茄钟交互、强密码生成、实时记账、Markdown 渲染、离线数据记录没有任何功能退化。
- **全动态视效**：完美的 `framer-motion` 切换、平移、阻尼物理粒子和流畅的点击反馈一并保留！
- **自适应系统主题**：支持夜间模式跟随系统无缝变色，高对比高雅色温。
- **低体积极速加载**：小程序安装包仅有数 KB，秒开且完全不占手机空间。

## 如何导入及在本地运行

1. 下载或导出整个 `/wechat-miniprogram` 文件夹到您的本地电脑。
2. 打开 **[微信开发者工具 (WeChat Developer Tools)](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)**。
3. 点击 **「导入项目」 (Import Project)**按钮。
4. 选择刚才导入的 `/wechat-miniprogram` 文件夹。
5. AppID 可以选择「测试号」 (Tourist Mode) 或填写您的真实微信小程序 AppID。
6. 点击确定即可。在左侧模拟器里，您能立即看到原生美观的 “春日小筑首屏启动页”，点击 **「一键开启春日小筑 🚀」** 即可瞬间载入高致、极致顺滑、零删减的精品完美体感。

## 微信小程序线上部署环境配置（重要）

当您准备将该小程序正式上传并发布至微信公共平台时，由于微信官方的安全域限制，您需要在配置后台完成以下简单步骤：

1. 登录您的 **[微信公众平台](https://mp.weixin.qq.com/)** 小程序管理后台。
2. 导航至 **「开发」 -> 「开发管理」 -> 「开发设置」**。
3. 滚动到 **「业务域名」 (Business Domain)** 模块。
4. 添入你所部署的春日小筑 Web 网站地址（即 `app.js` 中 `globalData.webUrl` 配置的在线主域名，如 `https://your-domain.com`）。
   > **提示：** 微信要求在该域名根目录下上传校验文件。在 Web 项目的 `public/` 目录下放置该校检文件（如 `WD_xxxxx.txt`），重新部署 Web 端，随后返回微信公众平台后台即可一秒验证通过！
5. 在 `wechat-miniprogram/app.js` 中，将 `webUrl` 换成你的正式访问域名。
6. 在微信开发者工具中点击右上角的 **「上传」** 提交审核。
